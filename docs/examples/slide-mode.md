# スライドモードサンプル

---

## スライドモードとは

スライドモードは、Markdownをプレゼンテーション形式で表示する機能です。

- フルスクリーンモードで利用可能
- スクロールでスライドを切り替え
- h1、h2、区切り線でスライドを分割

---

## 使い方

1. フルスクリーンボタンをクリック
2. スライドモードボタンをクリック
3. 上下にスクロールしてスライドを移動

---

# 新しいセクション

## 機能の詳細

### h1タイトル
- h1はタイトルだけのスライドになります
- 中央に大きく表示されます

### h2タイトル
- h2の上でスライドが区切られます
- セクションの開始を示します

### 区切り線
- `---` または `***` でスライドを区切ります
- コンテンツなしのスライド区切りとして機能します

---

## コード例

```javascript
function slideMode() {
  console.log('スライドモードが有効です！');
  return true;
}
```

コードブロックもスライド内で適切に表示されます。
