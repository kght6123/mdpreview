# スライドモードサンプル

## スライドモードとは

スライドモードは、Markdownをプレゼンテーション形式で表示する機能です。

- フルスクリーンモードで利用可能
- スクロールでスライドを切り替え
- h1、h2、区切り線でスライドを分割

## 使い方

1. フルスクリーンボタンをクリック
2. スライドモードボタンをクリック
3. 上下にスクロールしてスライドを移動

# 新しいセクション

## 機能の詳細

### h1タイトル
- h1はタイトルだけのスライドになります
- 中央に大きく表示されます

### h2タイトル
- h2の上でスライドが区切られます
- セクションの開始を示します

### 区切り線
- `---` または `***` でスライドを区切ります
- コンテンツなしのスライド区切りとして機能します

## コード例

```javascript
function slideMode() {
  console.log('スライドモードが有効です！');
  return true;
}
```

コードブロックもスライド内で適切に表示されます。

## リストの例

### 順序なしリスト
- 項目1
- 項目2
- 項目3

### 順序付きリスト
1. 最初のステップ
2. 次のステップ
3. 最後のステップ

# 区切り線の使い方

## 区切り線とは

区切り線（`---` または `***`）を使うと、h1やh2以外の場所でもスライドを区切ることができます。

### 使用例
- コンテンツの途中でスライドを分けたい時
- 空白のスライドを作りたい時
- 特定の内容を独立したスライドにしたい時

---
